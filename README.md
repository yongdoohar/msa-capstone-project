## 🧨 Attention
* 교육을 위한 재택근무가 아닌, 출근지에서 업무와 동시에 과제 수행시, Github 사이트로 Local에서 작성한 프로그램이 `Upload` 되지 않도록 주의
* Web Browser IDE 환경에서 과제 수행시, Client의 Network 상황에 따라 작업중인 내용이 유실될 있으니 `저장` 또는 `commit(push)`
* 과제수행 중 문의사항있을 경우, 카카오톡 오픈채팅창 이용
